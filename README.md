# 🏠 Smart Home Alerting System

A Smart Home Alerting System developed using **ESP8266 (NodeMCU)** and IoT technologies. The system monitors motion, door status, fire, and nearby objects using multiple sensors and provides real-time alerts through a web-based dashboard.

## ✨ Features

* Secure Login Dashboard
* Motion Detection (PIR Sensor)
* Door Open/Close Detection
* Fire Detection
* Ultrasonic Distance Monitoring
* Real-Time Alerts
* Event Logs with Timestamps
* LED & Buzzer Alerts
* Wi-Fi Based Monitoring

## 🛠️ Technologies Used

* ESP8266 (NodeMCU)
* Arduino IDE
* Embedded C++
* HTML
* CSS
* JavaScript
* ESP8266WebServer

## 🔧 Hardware Used

* ESP8266 NodeMCU
* PIR Sensor
* Reed Switch
* Ultrasonic Sensor (HC-SR04)
* Flame Sensor
* Buzzer
* LEDs

## ▶️ Getting Started

1. Clone this repository.
2. Open `fin.ino` in Arduino IDE.
3. Install the required ESP8266 board package and libraries.
4. Update the Wi-Fi credentials in the code.
5. Upload the code to the ESP8266 board.
6. Open the IP address shown in the Serial Monitor to access the dashboard.
